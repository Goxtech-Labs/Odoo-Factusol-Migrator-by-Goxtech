FactuSol Core

Capa de compatibilidad FactuSol para Odoo Community

Reproduce la lógica de FactuSol que no es nativa en Odoo: numeración TIPO-TIP-COD, triple descuento por línea, régimen de IVA por documento con tabla de traducción de impuestos, tarifas por margen y campos puente para migrar de forma idempotente. Base reutilizable por factusol_migrator.
